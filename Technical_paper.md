# Apache Kafka

## What is Apache Kafka?

**Apache Kafka** is an open-source event streaming platform designed to handle real-time data feeds with **high throughput** and **low latency**.

It is widely used for building real-time data pipelines and streaming applications.

---

## What is an Event?

An **event** represents an action performed by a user or system, such as:
- Adding an item to a cart
- Placing an order
- Making a payment
- Logging in to an application

Events capture *what happened*, *when it happened*, and *who or what triggered it*.

---

## Origin of Apache Kafka

Apache Kafka was originally developed by **LinkedIn in 2011**.  
It is built using **Java** and **Scala**.

Later, Kafka was donated to the **Apache Software Foundation**, where it became a fully open-source event streaming platform.

---

## Apache Kafka Ecosystem

Kafka integrates with many technologies, most of which are part of the **Apache Software Foundation**.

Some key components of the Apache Kafka ecosystem include:
- **Apache Spark**
- **Apache NiFi**
- **Apache Flink**
- **Apache Hadoop**
- **Apache Camel**
- **Apache Cassandra**

These tools help with stream processing, data ingestion, analytics, and storage.

---

## Kafka in the Real World

Today, Apache Kafka has evolved into one of the **most widely used event streaming platforms**.

Over **80% of Fortune 500 companies** use Kafka, including:
- Target
- Microsoft
- Airbnb
- Netflix

Kafka enables these organizations to deliver **real-time, data-driven customer experiences** at scale.
